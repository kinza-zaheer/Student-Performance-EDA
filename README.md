# 🎓 Student Performance Analysis (EDA)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on student academic performance using Python. It explores how different factors such as gender, parental education, marital status, and ethnicity influence student scores in Math, Reading, and Writing.

The goal is to uncover meaningful insights from educational data using data analysis and visualization techniques.

---

## 📊 Objectives
- Understand student performance patterns
- Analyze impact of demographic factors on scores
- Identify trends and relationships in data
- Visualize insights for better interpretation

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset Features
- Gender
- Ethnic Group
- Parent Education Level
- Parent Marital Status
- Weekly Study Hours
- Math Score
- Reading Score
- Writing Score

---

## 🔄 Data Processing Steps
- Loaded dataset using Pandas
- Handled missing values
- Removed unnecessary columns
- Cleaned inconsistent data
- Transformed study hours format

---

## 📈 Key Analyses Performed
- Gender distribution analysis
- Score comparison based on parental education
- Impact of marital status on performance
- Ethnic group distribution
- Detection of outliers using boxplots

---

## 📊 Visualizations Used
- Count Plot (Gender & Ethnic Groups)
- Heatmap (Correlation & averages)
- Boxplots (Score distribution)
- Pie Chart (Ethnic group distribution)

---

## 📌 Key Insights
- Parental education shows influence on student scores
- Score variations exist across ethnic groups
- Data contains some outliers in performance
- Gender distribution is fairly balanced
